What it does

Super Globals Plus is a developers' tool that makes evaluating variables easy. It produces a clear overview of the contents of strings, arrays, constants, objects, arrays within object, objects within arrays or whatever. Because the amount of information given by this tool was overwhelming to most new users of the Super Globals Plus script, version 1.24 and later only shows $_POST and $_SESSION by default.

Features:

  • Show all GLOBALS, or just the Superglobals you need to evaluate.
  • Show all defined constants (v1.2).
  • Show a list of included files (v1.22).
  • Show Superglobals either in a javascript popup window or at the end of each rendered page. (v1.4)
  • Optionally hide HTTP_ variables when showing all globals.
  • Show the whole content of each variable, including all objects and arrays etc!
  • Recursion protection (optional: recursion level/off)
  • Show the variable type (array, string, object, integer, NULL, boolean, resource).
  • Converts strings into their htmlentities before output (v1.0)
  • Clear output (see screenshot to the right).
  • Optional IP address protection.
  • Admin switches for all options.

How to use it

After you have logged into your store's admin and installed the plugin, you will see the Configuration :: Superglobals selection. Within that selection are the values that you can use to control the plugin's display.

Installation and Removal

This plugin is now encapsulated and has no core-file overwrites. Installation and removal are simplified.

Installing Super Globals

  1. Unzip this distribution's zip-file; you'll see a zc_plugins directory, and inside it a SuperGlobals directory, and inside that another directory with a version number.
  2. Copy the plugin's SuperGlobals directory to your site's file-system's zc_plugins directory.
  3. Log into your Zen Cart admin and navigate to Modules :: Plugin Manager. Click on Super Globals, then click Install and confirm by clicking Install.
  4. You'll see a Superglobals element under the admin's Configuration tab; proceed to the configuration step, see below.

Note: These files are automatically removed when v3.0.0 or later of this plugin is installed:

  1. /includes/auto_loaders/config.SuperGlobals.php
  2. /includes/classes/observers/SuperGlobalsObserver.php
  3. /includes/functions/extra_functions/superglobals.php
  4. /includes/templates/template_default/css/superglobals.css
  5. /YOUR_ADMIN/includes/auto_loaders/config.SuperGlobalsAdmin.php
  6. /YOUR_ADMIN/includes/functions/extra_functions/superglobals.php
  7. /YOUR_ADMIN/includes/languages/english/extra_definitions/superglobals.php

Un-install

Navigate to your Zen Cart admin's Modules :: Plugin Manager, highlight the entry associated with Super Globals and click the Un-Install button.
